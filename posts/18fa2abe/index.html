# 一。知识体系

  1. 功和能的关系
  2. 能的定义：如果一个物体具有做功的本领，那么物理学中就称这个物体具有能；
  3. 能是物体做功本领大小的量度；
  4. 做功是能量转移和转化的方式；
  5. 功是能量转移和转化的量度；
  6. 动能
  7. 动能的定义：物体由于运动而具有的能量
  8. 影响动能大小的因素：物体的质量和运动的速度。
  9. 动能的计算公式：

Ek=12mv2E_k = \frac{1}{2} mv^2Ek​=21​mv2

  1. 重力势能
  2. 重力势能的定义：物体处于某一高度时所具有的势能；
  3. 影响重力势能大小的因素：物体的质量和所处的高度；
  4. 重力势能的相对性和零势能面的选取；
  5. 重力势能的计算公式：

Ep=mghE_p = mghEp​=mgh

  1. 弹性势能
  2. 弹性势能的定义：物体发生弹性形变时所具有的能；
  3. 影响弹性势能大小的因素：弹性物体的劲度系数和形变程度；
  4. 弹性势能的计算公式：

E=12k∆x2E = \frac{1}{2}k∆x^2E=21​k∆x2

  2. 机械能的概念；
  3. 机械能转化的条件；(以蹦极为实例)
  4. 机械能转化过程中守恒的条件；
